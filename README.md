MamaPutOnline - Landing Page Project

Project Overview

MamaPutOnline is a professional landing page for an online Nigerian cooked meal delivery service. The website showcases authentic Nigerian dishes and soups, allowing customers to browse products and navigate through categories. This version is built exclusively with HTML and CSS, adhering to the requirement of no JavaScript.

Business Description

MamaPutOnline specializes in delivering authentic, home-cooked Nigerian meals prepared with traditional recipes and fresh ingredients. The business offers a wide variety of traditional Nigerian soups, rice dishes, swallows, proteins, and special combos.

How to Use

Opening the Website

1.
Open index.html in a web browser.

2.
The landing page will load with all sections visible.

Browsing for Products

1.
Use the sidebar on the left to navigate by category.

2.
Click on a category link to smoothly scroll to that section.

3.
Browse the product cards in each section.

Search Bar

•
The search bar is presentational. It allows text input and has a search icon, but does not dynamically filter products as JavaScript is not used.

Add to Cart Buttons

•
The "Add to Cart" buttons are presentational links. They do not add items to a cart or store data, as JavaScript is not used.
